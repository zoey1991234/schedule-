# schedule-

# 概要

基於html跟flask的學生分組與排班簡易網頁

# 環境配置

開發環境：Windows 10　，　Visual Studio Code

# 運行步驟

1.建立資料庫

2.安裝必要的依賴

`pip install Flask`

`pip install mysqlclient`


3.運行 Flask 應用

`python app.py`
